## Working rule

- Prefer these links over third-party summaries when checking syntax, rendering behavior, directives, or CLI capabilities.
- Use the Marp official site for product-level context.
- Use Marpit docs for markdown syntax, directives, image behavior, and theme CSS.
- Use the GitHub repositories for source-of-truth implementation details, release context, and CLI usage docs.
